# IEEE TechFest AR Camera Studio

Welcome to the official repository for the IEEE TechFest Sri Lanka 2025 AR Camera Platform, developed by Ceylon Labs. This project delivers a modern, interactive, and high-performance web-based AR camera experience for event participants and the broader tech community.

## ðŸ› ï¸ Getting Started

### 1. Clone the Repository
```sh
git clone https://github.com/SeneshFitzroy/ieee-techfest-camera-app.git
cd ieee-techfest-camera-app
```

### 2. Run Locally (Windows)
Double-click `start-server.bat` or run:
```sh
start-server.bat
```
Then open [http://localhost:8080](http://localhost:8080) in your browser.

### 3. Run Locally (Cross-platform)
Use any static server, e.g.:
```sh
python -m http.server 8080
```

### 4. Deploy
- Deploy to GitHub Pages, Vercel, Netlify, or any static hosting provider.
